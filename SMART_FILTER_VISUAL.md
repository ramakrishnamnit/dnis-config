## 11. Visual Examples by Use Case

### Example A: Date Range Filter
**Task**: Find records created in January 2025

**Setup:**
```
Condition 1: Created_Date [Greater than] [2024-12-31]
Condition 2: Created_Date [Less than]    [2025-02-01]
Logical Op:  AND
```

**Display:**
```
[AND] [Created_Date Greater than 2024-12-31]
      [Created_Date Less than 2025-02-01]
```

### Example B: Multi-Value Selection
**Task**: Show records from CC, GB, or RB business units

**Setup:**
```
Condition 1: Business_Unit [Equals] [CC]
Condition 2: Business_Unit [Equals] [GB]
Condition 3: Business_Unit [Equals] [RB]
Logical Op:  OR
```

**Display:**
```
[OR] [Business_Unit Equals CC]
     [Business_Unit Equals GB]
     [Business_Unit Equals RB]
```

### Example C: Text Search Across Columns
**Task**: Find "payment" in any description field

**Setup:**
```
Condition 1: Description     [Contains] [payment]
Condition 2: Service_Name    [Contains] [payment]
Condition 3: Notes           [Contains] [payment]
Logical Op:  OR
```

**Display:**
```
[OR] [Description Contains payment]
     [Service_Name Contains payment]
     [Notes Contains payment]
```
