## Color Scheme

### Inactive Button
- Border: `border-border` (gray)
- Text: `text-foreground` (normal)
- Background: `glass-hover` effect

### Active Button
- Border: `border-primary` (red/primary)
- Text: `text-primary` (red/primary)
- Badge: `variant="secondary"` with "Active" text

### Popover
- Background: `glass` effect
- Border: `border-border`
- Width: `320px` (80 in Tailwind units)
- Alignment: Right-aligned with button

## Accessibility Features

1. **Keyboard Navigation:**
   - Tab to button
   - Enter/Space to open popover
   - Tab through date inputs
   - Escape to close popover

2. **Screen Reader Support:**
   - Button labeled "Date Range"
   - Date inputs have proper labels
   - Active state announced

3. **Visual Indicators:**
   - Clear focus states
   - Active badge for filtered state
   - Color changes indicate interaction

## Animation & Transitions

1. **Popover:**
   - Smooth fade-in/out
   - Slide animation from button

2. **Button:**
   - Hover effect (glass hover)
   - Color transition when active
   - Badge fade-in when filter applied
